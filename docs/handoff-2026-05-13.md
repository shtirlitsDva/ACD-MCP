Stale `activeWorktreePath` field dropped.
- The `Application.Idle` auto-start hook is now **permanent under `#if DEBUG`** (commit `75ca8f0`, v20). DEBUG builds auto-open the pipe on first idle tick; Release builds keep the existing user-controlled behaviour. Self-removes on first fire; Terminate also `-=` for the unload-before-Idle case.
- Civil 3D licensing is one-instance-per-Windows-session. If the user has Civil 3D open visibly, a `/Automation` launch silently blocks on a license popup the agent can't see. Always check `Get-Process acad` first and match-by-PID to discriminate user processes from agent-launched ones.
- The G3/G4 path benefits from `Test-Path \\.\pipe\NAME` being unreliable for named-pipe readiness (returns false for valid-but-unconnected pipes). Use `[System.IO.Directory]::GetFiles("\\\\.\\pipe\\")` instead. Documented in the autonomous-bootstrap section.
- `src/Acd.Mcp/SharedAssemblies.Config.json` is now the tracked source-of-truth (commit `71f4ef9`); MSBuild copies it to `bin/Debug/` per build. Don't hand-edit the `bin/Debug/` copy — it's overwritten on `PreserveNewest`.
</misc-gotchas>
